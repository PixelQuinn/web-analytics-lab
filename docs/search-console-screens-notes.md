# Search Console Screens & Notes

## Purpose
This document captures the Search Console setup status, validation steps, and early SEO observations for the Web Analytics Lab project. It serves as a running record of how organic search performance is being monitored on my live portfolio site.

## Property Setup Notes
- Google Search Console was connected to the live portfolio site, [quinnreams.com](https://www.quinnreams.com/).
- The site is being monitored as part of a broader web and SEO analytics case study.
- Search Console is being used alongside GA4 to understand search visibility and organic traffic opportunities.

## What Search Console Adds to This Project
GA4 helps measure on-site behavior, while Search Console helps measure search performance before users arrive on the site.

Key questions Search Console helps answer:
- Is the site appearing in Google Search?
- What queries are generating impressions?
- Which pages are earning clicks?
- What SEO opportunities exist as more portfolio content is published?

## Metrics to Watch
The primary Search Console metrics for this project are:

- **Total Clicks** — how many times users clicked the site from Google Search
- **Total Impressions** — how many times the site appeared in search results
- **Average CTR** — the percentage of impressions that turned into clicks
- **Average Position** — the average ranking position of the site for queries and pages

## Early Notes
At the start of the project, Search Console data may be limited because:
- the site is still relatively new in its current form
- organic search impressions take time to accumulate
- the portfolio currently has limited indexed content compared with a larger website

Because of that, early Search Console insights may be directional rather than definitive.

## Planned Search Console Use
As more data accumulates, Search Console will be used to evaluate:

- whether branded search is appearing
- which portfolio pages earn impressions first
- whether project pages improve organic visibility
- whether page titles and descriptions are attracting clicks
- where future SEO improvements should be focused

## Expected Limitations
- Low search volume in the early stages
- Limited number of indexed pages at first
- Portfolio content still being expanded
- Search data may lag behind site updates

## Why This Matters for the Project
Search Console gives this project an SEO layer, not just an on-site analytics layer. Together, GA4 and Search Console help measure both:

1. how users find the site
2. what they do once they get there
